Current Issues

Currently this tool uses localstorage to store all posts, this doesn't work for users with a LOT of very long posts, as localstorage is limited to around 5mb in most browsers.
This tool uses FileSaver to prompt to download file as well as JSZip to zip the posts. All posts are currently saved in localStorage and the filedownload stored in ram.
